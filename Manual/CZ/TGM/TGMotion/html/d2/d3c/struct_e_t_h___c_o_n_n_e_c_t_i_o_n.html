◆ Command

int32_t ETH_CONNECTION::Command

Command (positive values) or error (negative values). PLC sets the command value, TGMotion executes command and either clears the value to 0 (idle) or sets an error.

Value Description
0 idle, the previous command has been completed
1 close connection, clears InBufferLastIndex. Should be called during PLC startup
2 open connection
10 send the data from outpubuffer with the length given by OutBufferDataLen
-2 connection cannot be established
-10 send data failed

◆ InBufferLastIndex

uint32_t ETH_CONNECTION::InBufferLastIndex

Index (offset) to the input circular buffer. When input data are received, this value points to the first byte after received data. Value is from 0 to the InBufferLen-1.

Note
Read only: value is set by TGMotion.
