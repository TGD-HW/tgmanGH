◆ pCNCExMemory

void* PLC_DATA::pCNCExMemory

pointer to CNCEx memory. Declared in tgm_cnc.h file as cnc::TGM_CNC_MEMORY. Here only void * is used to eliminate need of cross reference of the tgm_cnc.h file. PLC normally does not need this memory, it is set for advanced CNC control.
