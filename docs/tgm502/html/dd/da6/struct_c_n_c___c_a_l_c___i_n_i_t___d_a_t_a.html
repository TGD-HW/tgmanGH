Detailed Description

Initialization structure.

Holds pointers to useful functions: access to shared memory and output debug information to debug console. (dbgview.exe for Win DLL plug-in, TGM Debug Console for real-time plug-in). All the fields are set by TGMotion and used by plug-in.
